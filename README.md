## 原神助手

中文 | [English](README-en.md)

> 🚧 请注意：本项目仍在开发中。

### 介绍

一个为 _原神_ 玩家编写的小工具，基于 [`Electron`](https://www.electronjs.org/) 和 [`React`](https://reactjs.org/)。

> [_原神_](https://ys.mihoyo.com/) 是一款热门的动漫游戏，由 [米哈游](https://www.mihoyo.com/) 公司开发。
